# PR説明文: Marine Cleanup Robot Simulation 全体解説

## 概要

このPRは、`Marine Cleanup Robot Simulation` がどのようなシステムかを、初見のレビュー担当者でも理解しやすいように説明するための資料です。  
このプロジェクトは、海洋ごみを回収するロボットと海洋生物の動きをリアルタイムで可視化し、回収効率と生態系への影響を同時に観察できるシミュレーションです。

## このシステムでできること

- 海洋上に漂うごみの発生と移動をシミュレーションできる
- 複数種類のロボットが役割分担してごみを探索・回収できる
- 海洋生物がロボット接近に反応して回避行動やストレス変化を示す
- 画面上で状態をリアルタイムに可視化できる
- 回収量、衝突、ストレス、エネルギー消費などを指標として確認できる

## システム全体の構成

このアプリケーションは、バックエンドとフロントエンドで役割を分けた構成になっています。

### バックエンド

バックエンドは Python / FastAPI で実装されています。  
主な役割は以下です。

- シミュレーション状態の管理
- エージェントの行動更新
- 各種設定値の受け取り
- スナップショットや統計情報の生成
- WebSocket 経由でフロントエンドへ状態を配信

特に `backend/app/simulation/engine.py` がシミュレーション更新の中心です。

### フロントエンド

フロントエンドは React / TypeScript で構成されています。  
主な役割は以下です。

- シミュレーションの開始・停止・設定変更
- Canvas 上でのロボット、ごみ、海洋生物の描画
- 統計情報の表示
- バックエンドから受け取った状態のリアルタイム反映

描画は `frontend/src/renderers/`、UI は `frontend/src/components/` に分かれています。

## 登場する主なエージェント

### Scout

- ごみを探索して発見する役割
- 発見した情報を共有し、回収の起点を作る

### Collector

- ごみを回収する役割
- 回収後は基地へ戻って搬入する
- バッテリー残量も考慮して行動する

### Marine Life

- 周辺の状況に応じて移動する海洋生物
- ロボットが近づくと回避行動を取る
- 接近の影響をストレス値として蓄積する

### Trash

- 海上を漂流する回収対象
- ロボットの探索・回収対象として扱われる

## 画面上で確認できること

このシミュレーションでは、単にごみを集めるだけでなく、回収活動の副作用も観察できます。

- どれだけごみを回収して基地へ搬入できたか
- ロボット同士や対象との接触がどれくらい起きたか
- 海洋生物へのストレスがどれだけ増えたか
- ロボットの残エネルギーがどの程度維持されているか

これにより、回収効率だけでなく、環境負荷とのバランスも評価できます。

## 主な技術スタック

- Backend: Python 3.12 / FastAPI / WebSocket
- Frontend: React 19 / TypeScript / Tailwind CSS / Canvas API
- Infra: Docker / Docker Compose
- Verification: unittest / GitHub Actions

## 主なファイルと責務

- `backend/app/simulation/engine.py`
  - シミュレーション更新の中核処理
- `backend/app/simulation/agents/`
  - Scout / Collector / Marine Life / Trash の行動定義
- `backend/app/models/schemas.py`
  - 設定値、スナップショット、スコアのデータ契約
- `backend/app/api/`
  - REST API と WebSocket エンドポイント
- `frontend/src/components/`
  - UI コンポーネント
- `frontend/src/renderers/`
  - Canvas 描画ロジック

## このPRで伝えたいポイント

このプロジェクトは、海洋ごみ回収の効率だけを追うのではなく、海洋生物への影響も同時に観察できる点が特徴です。  
ロボットの協調行動、回収効率、エネルギー管理、生物ストレスを一つの画面で確認できるため、シミュレーションとしての学習用途や検討用途に向いた構成になっています。
