# Program Docs

このディレクトリは、実装済みプログラムの詳細仕様を保持する。  
`documents/` が実装判断の要約であるのに対し、`program_docs/` は runtime behavior、API 契約、Docker / テスト / CI の運用を詳しく記載する。

## 文書一覧

- `runtime_behavior.md`: シミュレーション挙動、役割、スコア、基地、再生成
- `api_contracts.md`: Config / Snapshot / Event の契約
- `development_workflow.md`: Docker、ローカル実行、テスト、CI
